Web Semântica

A web semântica é um conceito que visa organizar os dados na web, permitindo que as máquinas os compreendam e interajam com ela (WEB). Para com isso usamos várias tags semânticas que facilitaram o trabalho dos mecanismos de busca.

Neste exemplo estamos usando a web semântica na prática, use um bloco de notas para acessar o documento html e ver como essa página foi contruída.
